# ❤️ Heart Disease Prediction System

## 📌 Overview  
This project is a machine learning-based system that predicts the likelihood of heart disease in a patient using medical attributes. It helps in early detection and supports better clinical decision-making.

---

## 🚀 Features  
- Predicts heart disease based on user input  
- Data preprocessing and feature selection  
- Multiple machine learning models implementation  
- Accuracy comparison of models  
- Simple and user-friendly interface  

---

## 🧠 Technologies Used  
- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Matplotlib / Seaborn  
- Streamlit  

---

## 📊 Dataset  
- Contains medical attributes like age, sex, chest pain type, blood pressure, cholesterol, etc.  
- Dataset is cleaned and preprocessed for better model performance  

---

## ⚙️ Working Process  
1. Data Collection  
2. Data Preprocessing  
3. Feature Selection  
4. Model Training  
5. Model Evaluation  
6. Prediction  

---

## 🤖 Machine Learning Models Used  
- Logistic Regression  
- Decision Tree  
- Random Forest  
- Support Vector Machine (SVM)  

---

## 📈 Results  
- Achieved good accuracy across models  
- Random Forest and Logistic Regression performed best
